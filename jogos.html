Jogo 1 - Aventura Matemática

Embarque em uma jornada emocionante através de mundos virtuais enquanto desafia sua mente com problemas matemáticos desafiadores.

Instruções:

Use as setas do teclado para mover o personagem. Utilize a barra de espaço para pular. Resolva os problemas matemáticos para avançar de nível.

Requisitos do Sistema:

Windows 10, processador Intel Core i5, 8GB de RAM, placa de vídeo dedicada.

Jogo 2 - Exploração Científica

Descubra os segredos do mundo natural enquanto realiza experimentos científicos interativos e emocionantes.

Instruções:

Use o mouse para interagir com os elementos do jogo. Siga as instruções na tela para realizar experimentos.

Requisitos do Sistema:

macOS Catalina, processador Intel Core i7, 16GB de RAM, placa de vídeo AMD Radeon Pro.

Jogo 3 - Mistérios Históricos

Explore a riqueza da história e da cultura de civilizações antigas enquanto desvenda enigmas desafiadores e intrigantes.

Instruções:

Use o teclado e o mouse para explorar os cenários. Resolva os quebra-cabeças para desbloquear novas áreas.

Requisitos do Sistema:

Linux Ubuntu 20.04, processador AMD Ryzen 5, 12GB de RAM, placa de vídeo NVIDIA GeForce GTX.
